蹴鞠けまりストラックアウトとは？

今は昔、ここ平安の地では貴族たちが余暇を持て余していた、
かの者たちは仕事が少なく、１日の多くを遊びに費やしていた、
しかし、この貴族たちは、飽きが早くどの遊びも長くは続かなかった、

ただ１つを除いては、、、、

それが、鞠会まりかいであった、
月に１度開かれるそれは鞠を蹴るだけであったが貴族の心を大きく惹きつけた、


ある時、貴族の１人が鞠を使った新たな遊戯を、思いついた、
区切られた、木の枠に障子を貼り撃ち抜くというその遊びは当時の人達にとって斬新な物であった
すぐにそれは流行し、いつしか技術を競い合うまでにまでなっていた、

これが蹴鞠ストラックアウトの起源とされている、

さて、今日はこの遊戯を庶民が体験できる日
どのような人が来て、どのような技が発揮されるのか？ それは誰にもわからない、、、、、
